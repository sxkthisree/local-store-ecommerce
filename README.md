# 🛒 Local Store E-Commerce Platform

A modern and responsive e-commerce website for a local grocery store, developed using HTML, CSS, JavaScript, Node.js, and Express.js.

## 📌 Project Overview

This project enables customers to browse products and add items to a shopping cart through a clean and user-friendly interface. It simulates a basic online shopping experience for a local store.

## ✨ Features

* Product Listings
* Product Names, Descriptions, and Prices
* Shopping Cart Functionality
* Responsive User Interface
* Modern Card-Based Product Design
* Easy Navigation and User Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/sxkthisree/local-store-ecommerce.git
```

2. Navigate to the project folder

```bash
cd local-store-ecommerce
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

5. Open in browser

```text
http://localhost:5000
```
